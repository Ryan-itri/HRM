# 派外單位人員出勤管理辦法 (草案)

## 1. 目的
為有效管理派外單位人員之出勤、差旅與休假，特制定本辦法，以作為系統設定與行政管理之依據。

## 2. 適用對象
凡本公司派駐外部單位之所有技術人員與行政人員。

## 3. 工作時間與出勤規定

### 3.1 彈性上下班制
- **上班時間**：08:00 ~ 09:00 (彈性區間)
- **下班時間**：17:00 ~ 18:00 (依上班時間往後推算 9 小時，含休息時間)
- **每日工時**：正常工作時間為 8 小時。

### 3.2 休息時間
- **午休時間**：12:00 ~ 13:00 (共 1 小時)，不計入工時。

### 3.3 簽到退機制 (配合系統)
- **簽到**：
    - 每日抵達派駐地點後，必須透過「現場 Kiosk 看板」掃描 QR Code 進行簽到。
    - 系統將依據簽到時間判定是否遲到。
    - *判定標準*：09:01 後簽到者視為遲到。
- **簽退**：
    - 每日離開派駐地點時，必須掃描 QR Code 進行簽退。
    - 系統將依據簽退時間與當日上班時間，計算當日工時是否滿足 8 小時。

## 4. 請假與缺勤

### 4.1 請假程序
- 若需請假，應事前於系統填寫請假單並於線上系統登載 (掃描 QR Code 後選擇「請假」或「外出」)。
- 緊急病假得於當日 09:00 前完成通報或系統登錄。

### 4.2 狀態定義 (系統選項)
- **上班**：正常執行勤務。
- **下班**：結束當日勤務。
- **外出**：因公務暫時離開駐點 (需預計返回時間)。
- **出差**：奉派至其他地點執行任務 (非原駐點)。
- **請假**：包含特休、事假、病假等。

## 5. 加班規定
- 延長工時 (加班) 需事前經主管核准。
- 加班之起算點為當日正常工時結束後。

## 6. 異常處理
- 若因設備故障無法掃碼，應立即向管理人員回報，並於事後由管理端補登紀錄。
